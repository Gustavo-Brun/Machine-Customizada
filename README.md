## Getting started

This API contains the '/convert' route, which returns a specific number converted to the chosen numeric base.

### URLs

- **Base** — https://machine-customizada.onrender.com

### cURL structure

- **-X** — Method + URL
- **-H** — Headers
- **-d** — Data

### Request Example

- `curl -X POST https://machine-customizada.onrender.com/convert -H "Content-Type: application/json" -d "{\"from\": 10, \"to\": 16, \"value\": 1950}"`
